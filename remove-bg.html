Remover fundo

Tira o fundo das suas fotos automaticamente. Em lote ou uma a uma — IA roda local no navegador.

⚠ Aviso: esta ferramenta não funciona quando o arquivo é aberto direto do explorador (file://). A IA precisa de Fetch API + Web Workers, que o Chrome bloqueia nesse modo.
Solução: use o site no GitHub Pages (https://betopr.github.io/marca-dagua/remove-bg.html) ou rode um servidor local.

2. Qualidade

Rápido: ~3s/imagem, modelo menor (~40MB).
Qualidade: ~8s/imagem, melhor resultado (~80MB).

3. Como funciona

A primeira vez baixa o modelo de IA (~40-80MB dependendo da qualidade). Depois fica cacheado e processa em segundos. Tudo offline: nenhuma imagem sai do seu navegador.
